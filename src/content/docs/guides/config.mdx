The configuration file is a [TOML](https://toml.io) file that specifies options also available on the command line.

## Location

By default, lychee looks for `lychee.toml` in the current working directory:

```bash
lychee --config ./path/to/config.toml
```

<Aside type="tip">
  CLI options take precedence over configuration file settings.
</Aside>

## `.lycheeignore` File

You can create a `.lycheeignore` file to exclude specific links:

```text title=".lycheeignore"
# Exclude LinkedIn (often blocks bots)
https://linkedin.com

# Exclude local development URLs
http://localhost
http://127.0.0.1

# Regex patterns work too
https?://.*\.local
```

<Aside type="note">
  The `.lycheeignore` file is useful for excluding links that frequently change or are intentionally broken (like examples).
</Aside>
