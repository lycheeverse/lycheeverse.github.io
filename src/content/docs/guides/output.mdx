Lychee provides flexible output options. You can customize how results are displayed in the terminal and how reports are saved to files.

## Display Modes (`--mode`)

Display modes control how lychee presents results in your terminal.

```bash
lychee --mode <MODE> <inputs>...
```

Available modes: `color`, `plain`, `emoji`, `task`

### Color Mode (Default)

```bash
lychee --mode color README.md
```

![Color Mode](./images/color.avif)

With verbose output:

```bash
lychee --mode color -vv README.md
```

![Color Mode Debug](./images/color-debug.avif)

### Plain Mode

```bash
lychee --mode plain README.md
```

![Plain Mode](./images/plain.avif)

Useful for automated scripts or piping output:

```bash
lychee --mode plain . | grep "FAILED"
```

### Emoji Mode

```bash
lychee --mode emoji README.md
```

![Emoji Mode](./images/emoji.avif)

### Task Mode

```bash
lychee --mode task README.md
```

## Output Formats (`--format`)

Output formats control how lychee writes reports when using `--output`.

```bash
lychee --format <FORMAT> --output <FILE> <inputs>...
```

Available formats: `compact`, `detailed`, `json`, `markdown`, `raw`

### Compact Format (Default)

```bash
lychee --format compact --output report.txt README.md
```

### Detailed Format

```bash
lychee --format detailed --output report.txt README.md
```

### JSON Format

```bash
lychee --format json --output report.json README.md
```

Useful for CI/CD pipelines and custom reporting tools.

### Markdown Format

```bash
lychee --format markdown --output report.md README.md
```

Useful for creating GitHub issues or documentation reports.

### Raw Format

```bash
lychee --format raw --output report.txt README.md
```

## Combining Options

```bash
# Show emoji in terminal, save JSON report
lychee --mode emoji --format json --output report.json README.md

# Use plain mode with detailed format
lychee --mode plain --format detailed --output report.txt README.md
```

## Examples

### CI/CD

```bash
lychee --no-progress --format json --output report.json .
```

### GitHub Actions

```yaml
- name: Link Checker
  uses: lycheeverse/lychee-action@v2
  with:
    format: markdown
    output: lychee-report.md
```
